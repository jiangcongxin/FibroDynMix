# Public Real-Data Smoke Figure

## Figure-Level Claim

This figure demonstrates that FibroDynMix can execute end-to-end on public raw
UMI fibroblast count data:

```text
raw public count matrices
-> marker-prior coverage audit
-> NB objective fit
-> state composition
-> FPI
-> cross-sectional transition flow
```

## Claim Boundary

```text
This is a technical smoke-test figure, not a disease-mechanism result.
```

The public data contain pooled mouse breast fibroblast/CAF count matrices from
DOI `10.6071/M3238R`. Biological interpretation would require full QC,
replicated sample-level design, cell curation, sensitivity analysis, and
external validation.
