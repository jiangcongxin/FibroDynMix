# Figure 1 Architecture and Evidence Map

## Figure-Level Claim

Figure 1 positions FibroDynMix as a raw-count generative mixture framework:

```text
raw UMI counts
-> negative-binomial likelihood
-> latent fibroblast state simplex
-> hierarchical study/donor-effect optimization
-> weak marker-prior orientation
-> uncertainty and transition-flow outputs
```

## Claim Boundary

```text
This is an architecture and evidence-map figure. It is not a standalone
biological validation figure and does not claim complete Bayesian VI yet.
```

## Panels

- **A. Generative FibroDynMix structure**
  - Schematic of model components and output layers, including study- and
    donor-level effect optimization.

- **B. Raw-count likelihood layer**
  - Compact model-term table distinguishing the approach from normalized
    gene-set scoring.

- **C. Evidence map**
  - Implemented, partial, and planned evidence layers across simulation,
    calibration, public smoke testing, transfer testing, and claim-boundary
    status.

- **D. Completed analysis packages**
  - Current manuscript evidence backbone: simulation benchmark, study-effect
    calibration, bootstrap uncertainty, logistic-normal VI posterior skeleton,
    transition flow, public raw-count smoke run, leave-study-out transfer
    benchmark, and public real-data transfer smoke.

## Role in the Manuscript

This figure should open the method/results section by making the modeling
scope explicit before the reader reaches simulation and real-data smoke
figures. It also prevents overclaiming by showing which components are already
implemented and which remain future work.
