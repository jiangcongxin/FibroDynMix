# Supplementary Figure S4. scVI-inclusive core NB objective-RMSE tradeoff

**A.** scVI-inclusive core benchmark points comparing NB objective,
latent-state RMSE, and downstream utility for NB-family methods across four
simulation scenarios and `n_outer = 10` or `20`.
**B.** Paired `n_outer = 10` to `20` contrasts showing that objective
improvement alone is not a reliable selector for latent-state recovery in the
five-replicate scVI-inclusive rerun.

scVI is included as a bounded latent baseline and projected to state-aligned
simplex features for comparison. This figure supports the optimizer-selection
claim; it is not a full deep generative benchmark or biological validation.
