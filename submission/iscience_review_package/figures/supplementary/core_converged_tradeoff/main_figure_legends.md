# Supplementary Figure S3. Core NB objective-RMSE tradeoff under higher outer-iteration budgets

**A.** Core NB-family benchmark points comparing negative-binomial objective,
latent-state RMSE, and downstream utility across four simulation scenarios,
two NB-family methods, and `n_outer = 10` or `20`.
**B.** Paired `n_outer = 10` to `20` contrasts showing cases in which lower
training NB objective coincided with worse latent-state RMSE.

This figure supports validation-aware model selection for the current
FibroDynMix optimizer. It is a bounded simulation optimizer-sensitivity figure
and does not constitute biological validation.
