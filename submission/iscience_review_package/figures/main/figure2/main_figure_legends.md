# Figure 2. Public raw-count FibroDynMix demonstration

**A.** Balanced normal and disease public raw-count fibroblast subset used for the demonstration.
**B.** Negative-binomial objective values during FibroDynMix fitting.
**C.** Disease-minus-normal shifts in fitted state composition.
**D.** Condition-level mean latent state composition.
**E.** Cell-level Fibroblast Plasticity Index (FPI) by condition.
**F.** Cross-sectional normal-to-disease state-flow matrix inferred from fitted compositions.
**G.** Cell-level latent state-mixture heatmap ordered by condition and dominant state, with the top strip indicating condition order.

The data are public mouse pooled fibroblast/CAF raw UMI counts from DOI 10.6071/M3238R. This figure demonstrates real raw-count execution and should not be interpreted as a disease mechanism without full QC, replicated donor design, and validation.
