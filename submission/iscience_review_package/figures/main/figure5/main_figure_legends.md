# Figure 5. External reproducibility across independent fibroblast disease datasets

**A.** Donor-level FibroDynMix state composition in GSE167339 Human1/Human2 control and perturbed collagen scar-system conditions.
**B.** GSE167339 leave-donor-out transfer convergence. Human3 is treated as hash-unknown donor robustness evidence because treatment labels are not publicly mapped.
**C.** Dominant-state fractions in external scar cohorts GSE156326 and GSE181316 after final-branch transfer, interpreted as descriptive ECM-rich disease-context patterns.
**D.** Disease-control scar module-score deltas in scRNA-seq pseudobulk validation. Point size encodes AUC; labels show disease-control sample counts and AUC. Multiple-testing-adjusted q values are retained in the source data and are not FDR-significant.

The figure supports bounded external reproducibility and descriptive disease-context biological utility, not diagnostic performance, causal mechanism, or therapeutic efficacy.
