# Figure 6. Cross-sectional state-flow inference and fibroblast plasticity index

**A.** Normal and disease condition-level fibroblast state composition in the rare-transition simulation.
**B.** Cross-sectional inferred normal-to-disease state-flow matrix estimated by entropy-regularized optimal transport using state-program transcriptional cost.
**C.** Cell-level Fibroblast Plasticity Index (FPI) by condition.
**D.** FPI distribution in simulator-labeled rare transition cells and other cells, with subset sizes shown on the x axis.

State flow is inferred from cross-sectional composition and state-program cost. It should not be interpreted as lineage tracing or observed temporal transition.
