# Figure 1. FibroDynMix model architecture and evidence boundary

**A.** Schematic of the FibroDynMix raw-count state-mixture model, linking UMI counts to a negative-binomial likelihood, continuous simplex state weights, study/donor adjustment, weak marker orientation, uncertainty intervals, and state-level summaries.
**B.** Model terms and manuscript readouts that distinguish likelihood-based state-mixture inference from normalized marker scoring alone.
**C.** Evidence spine linking each main result figure to the claim it supports and the boundary kept around that claim.
**D.** Claim boundary contract separating what is supported in the current manuscript from bounded interpretations and endpoints not claimed.

This figure defines the model and manuscript claim structure. It does not claim causal fibroblast transitions, clinical biomarker readiness, or a complete hierarchical variational posterior.
