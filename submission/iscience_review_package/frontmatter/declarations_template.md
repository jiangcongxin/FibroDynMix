# Declaration and Metadata Templates

## Declaration of Interests

Template if there are no competing interests after author confirmation:

The authors declare no competing interests.

Template if competing interests exist:

[Author initials] reports [relationship, company/institution, financial or
non-financial nature of interest]. All other authors declare no competing
interests.

The final manuscript statement must match the Cell Press declaration of
interests form.

## Funding Statement

Template:

This work was supported by [funder name] [grant number] to [author initials].

Template if no funding supported the work:

This research received no specific grant from any funding agency in the public,
commercial, or not-for-profit sectors.

Use only after author confirmation.

## Data Availability

Current manuscript-ready statement:

Public mouse breast fibroblast count data are available from DOI
10.6071/M3238R. Public human analyses used GSE246215, GSE167339, GSE156326, and
GSE181316-derived inputs as documented in the analysis manifests. Figure source
data and manifests are included with the submission review package.

Final upload requires:

- Public repository or archive link for generated source data and scripts, if
  the local workspace is not the final public access point.
- Permanent code repository or archival DOI for the FibroDynMix package.

## Code Availability

Current manuscript-ready statement:

FibroDynMix is organized as an R package. The review package includes the
manuscript-facing figure source data, manifests, legends, and export QC.

Final upload requires:

- Public code repository URL or archive DOI.
- Release tag or commit identifier.
