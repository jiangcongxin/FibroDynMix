# eTOC Blurb

## Primary Version

FibroDynMix infers fibroblast state mixtures directly from raw UMI counts in one
auditable workflow, and shows that a lower training objective can worsen state
recovery, motivating validation-aware selection that generalizes across public
fibroblast datasets through transfer and marker-gradient recovery.
