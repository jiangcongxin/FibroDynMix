# Supplementary Figure S2. Raw-count FibroDynMix execution on public fibroblast scRNA-seq data

**A.** Retained weak-prior fibroblast state markers after preparing the public raw UMI count matrices.
**B.** Negative-binomial objective values from the public-data smoke fit.
**C.** Public dataset and smoke-test scope summary.
**D.** Condition-level mean FibroDynMix state weights.
**E.** Cell-level Fibroblast Plasticity Index (FPI) by condition.
**F.** Cross-sectional normal-to-disease state-flow matrix inferred from fitted state compositions.

The public dataset is the Hum/Sebastian breast fibroblast raw-count record (DOI 10.6071/M3238R). This figure demonstrates end-to-end execution on public raw counts; disease-mechanism interpretation would require full real-data QC, replication, and sensitivity analysis.
